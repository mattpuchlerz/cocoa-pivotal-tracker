cocoa-pivotal-tracker by mattpuchlerz

Re-skins Pivotal Tracker to appear more like a Cocoa application.

Install

First, install GreaseMonkey for Firefox or GreaseKit for Safari. Then go the Cocoa Pivotal Tracker userscript and install it. Done.
